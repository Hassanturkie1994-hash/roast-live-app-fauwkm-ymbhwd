## â“ Troubleshooting

### VIP Club

**Q: I don't see the toggle to enable VIP Club**
- A: VIP Club enable/disable is in Settings, not Pre-Live Setup

**Q: Members list is empty**
- A: You may not have any VIP members yet, or they may have canceled

**Q: Can't find a specific member**
- A: Use the search bar in the Members modal

### Face Effects

**Q: Effect isn't showing**
- A: Make sure you selected an effect (not "None")
- Check that camera permissions are granted

**Q: Effect is laggy**
- A: Close other apps to free up GPU resources
- Restart the app if needed

### Color Filters

**Q: Camera disappeared when I applied a filter**
- A: This is a bug - please report it immediately
- Restart the app and try again
- Use lower intensity (50%)

**Q: Filter looks too strong**
- A: Reduce intensity using the slider
- Try a different filter

**Q: Filter doesn't match preview**
- A: Lighting affects how filters appear
- Adjust room lighting for best results
